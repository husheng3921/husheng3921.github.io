背景

最近折腾博客，开始采用github page方式的静态博客，但是考虑到国内访问GitHub速度慢且不稳定，故在原有GitHub和hexo(AirCloud搭建的博客基础上，迁移到码云Gitee上，这里有些坑提出来，供大家参考。参考博客：基于Gitee+Hexo搭建个人博客

环境搭建

打开hexo官网hexo doc;参照指南安装好node，git。注意国内访问npm镜像较慢，切换成淘宝镜像源，执行下面命令：
切换淘宝镜像
$ npm config set registry https://registry.npm.taobao.org
查看是否切换到淘宝镜像
$ npm info underscore

1、安装hexo

使用npm安装hexo
$ npm install -g hexo-cli
这里可能会出错，参考链接中的错误，cmd中输入命令行：
$ set-ExecutionPolicy RemoteSigned 选择A

    坑一
    在gitee上新建仓库时注意仓库名字要与你的Gitee账户一致，例如我的仓库名字husheng3921(与账户名一样),这样才能保证Gitee创建博客的域名就是http://husheng3921.gitee.io,否则后面会带有后缀，例如仓库名:blog，系统会生成http://husheng3921.gitee.io/blog,你还需要在_confiy.yml中root属性配置blog值。 点击博客仓库-服务-Gitee Pages
